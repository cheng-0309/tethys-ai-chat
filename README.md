# Tethys - AI Chat Assistant

A clean and minimal React application for an AI chat assistant.

## Project Structure

```
src/
├── components/
│   ├── Header.js
│   └── ChatContainer.js
├── pages/
├── styles/
│   ├── index.css
│   ├── App.css
│   ├── Header.css
│   └── ChatContainer.css
├── App.js
└── index.js
```

## Getting Started

### Installation

```bash
npm install
```

### Running the Application

```bash
npm start
```

The app will open at `http://localhost:3000`

### Building for Production

```bash
npm build
```

## Features

- Clean, minimal React structure with functional components
- Responsive design that works on desktop, tablet, and mobile
- Modern gradient header with app title
- Placeholder for chat interface
- Well-organized folder structure

## Technologies Used

- React 18
- CSS3 with media queries
- Create React App
